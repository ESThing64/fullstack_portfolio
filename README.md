# John Coakleys Fullstack Portfolio

## Usage

### Windows:
```
    git clone https://github.com/ESThing64/fullstack_portfolio.git
    cd fullstack_portfolio
    index.html
```
### iOS:
```
    git clone https://github.com/ESThing64/fullstack_portfolio.git
    cd fullstack_portfolio
    open index.html
```
